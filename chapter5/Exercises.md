# Exercises

### 1. Improve on the code in Quiz Question 5, using parentheses to provide clarity.
```
int result = number << 1 + 5 << 1
```
#### Answer:
```
int result = number << ((1 + 5) << 1)
```

### 2. What is the value stored by the following expression:
```
int result = number << 1 + 5 << 1
```
#### Answer:
```
The result is number * 4096
```
